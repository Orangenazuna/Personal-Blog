首页从零开始网站建立

由于网站是静态页面，如果想实现动态效果，就需要引入外部的插件，对使用到的和了解到的插件进行整理。

# 搜索功能配置 Algolia

Algolia 提供每个月 10000 次的免费搜索额度，基本上是够用了。
Algolia 提供了亚洲区域的服务器，覆盖了中国大陆以及其他亚洲地区。

Algolia

Leading Search & Discovery API.

# 评论系统 Waline

需要服务端云部署，为了国内用户也能访问推荐使用 Zeabur 部署。

Waline

快速，简单且高效的评论系统。

Zeabur

提供高品质软件开发服务的技术公司。

# 访问统计 Busuanzi

基于 Golang + Redis 的简易访问量统计系统。
++ 国内大佬为爱发电 ++

Busuanzi

用于统计网站访问量的 JavaScript 插件。
